# 📰 BBC News Classification –    Multi-Class Text Classification

## 📌 Project Overview
This project focuses on building a machine learning model to automatically classify BBC news articles into predefined categories such as business, sports, politics, technology, and entertainment.

It is a Natural Language Processing (NLP) project that demonstrates text preprocessing, feature extraction, and supervised machine learning techniques.

---

## 🎯 Objective
The objective of this project is to:

- Automatically categorize news articles into multiple classes
- Compare different classification algorithms
- Evaluate model performance using appropriate metrics
- Demonstrate practical NLP pipeline implementation

---

## 📊 Dataset
The dataset consists of BBC news articles grouped into the following categories:

- Business
- Sports
- Politics
- Technology
- Entertainment

Each article contains raw text data and its corresponding category label.

## 🔎 Methodology

### 1️⃣ Data Preprocessing
- Text cleaning
- Lowercasing
- Removing punctuation
- Stopword removal
- Tokenization

### 2️⃣ Feature Extraction
- Bag of Words (CountVectorizer)

### 3️⃣ Model Building
Different classification models were tested, such as:
- Naive Bayes
- Logistic Regression
- Support Vector Machine (SVM)

### 4️⃣ Model Evaluation
Models were evaluated using:
- Accuracy
- Precision
- Recall
- F1-score
- Confusion Matrix
